🏏 Virat Kohli Performance Dashboard

A comprehensive cricket analytics dashboard focused on analyzing the career batting performance of Virat Kohli across multiple years using statistical analysis and interactive visualizations.

The dashboard provides insights into batting consistency, scoring trends, strike rates, centuries, boundary analysis, and overall career progression. This project transforms raw cricket statistics into meaningful visual stories that help understand the evolution of one of the greatest modern-day batsmen.

📌 Project Overview

The Virat Kohli Performance Dashboard is designed to explore and visualize year-wise batting statistics from Test cricket performances.

Using data analytics and dashboarding techniques, the project highlights:

Batting consistency over the years
Performance trends and peak seasons
Century and half-century analysis
Strike rate and scoring efficiency
Boundary scoring patterns
Dot ball percentage analysis
Career milestones and records

This dashboard is ideal for cricket analysts, sports enthusiasts, and data visualization learners.

🎯 Project Objectives

The major objectives of this project are:

Analyze yearly batting performance of Virat Kohli
Track runs scored across seasons
Compare batting average and strike rate trends
Identify peak performance years
Study consistency using 50s and 100s
Analyze boundary-scoring patterns
Visualize career growth and fluctuations
Build an interactive sports analytics dashboard
